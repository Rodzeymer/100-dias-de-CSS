# 100-dias-de-CSS
https://rodzeymer.github.io/100-dias-de-CSS/

Como um dos desafios de aprendizagem, recomendo o https://100dayscss.com/, que apresenta 100 desafios de CSS puro, e muitos dos seus pré processadores, 
o desafio não é apenas replicar o exercício proposto mas incrementar ou modificar o código e visualizar o resultado, que no final é um tipo de arte.

É incrível como o CSS é poderoso, e com os pré processadores o limite é sua imaginação.

O próprio site recomenda, e eu endosso, o site codepen.io, que permite criar pequenos projetos web, com suporte para os já conhecidos HTML/CSS/JS,
permite o uso de scripts de JS externos, os pré processadores de CSS e de HTML, dando uma maleabilidade para o seu projeto, permitindo ir muito além
do convencional, é uma "rede social" onde você pode compartilhar projetos e pesquisar projetos de pessoas do mundo todo, um que eu gostei muito, para
demonstrar o poder dessa ferramenta, é o 3D 20d, https://codepen.io/vicentemundim/details/nXNvBw, um dado de 20 lados, em que você pode rolar o dado e
ele roda o dado para um número aleatório, vc pode copiar o projeto para realizar alterações e personalizar!

Enfim, essa é a minha coleção de 100 projetos de CSS, fiquem à vontade e vamos em frente!
